## üõ†Ô∏è How to Use cornell-marginalia

Once installed, use cornell-marginalia to create Cornell notes like this:

### Creating a New Cornell Note

- Press the shortcut **Ctrl+Alt+C** (Windows/Linux) or **Cmd+Option+C** (macOS) to create a new Cornell Note template.
- The template divides the note into three parts:
  - **Cue Column:** For keywords or questions.
  - **Note-taking Column:** For detailed notes.
  - **Summary Section:** For a brief summary of the note.

### Adding Cues and Marginalia

- Click inside the cue column to add questions or keywords that help in review.
- Use the note column for lecture notes or reading points.
- Add margin notes by right-clicking anywhere in the note and choosing **Add Marginalia** to highlight supporting ideas.

### Reviewing Your Notes

- Use the plugin‚Äôs review mode (activate from the plugin menu) to see cue questions only.
- Test your memory by answering questions before revealing full notes.
- Summaries display automatically at the bottom to recap important points.
